# jointlab-web

Assume you have installed npm and other necessary config.

```
git clone git@github.com:Rickyoung221/jointlab-web.git
```
```
cd jointlab-web/client
```
Then to start the web page and develop, run
```
cd client
npm install
npm start
```
Similarly, run the server:
```
cd server
npm install
npm start
```
